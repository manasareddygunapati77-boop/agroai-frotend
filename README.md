# AgroAI - Real-Time AI Crop Advisory System

## Overview
AgroAI is an AI-powered web application that helps farmers by providing:
- Crop recommendation
- Disease detection
- Fertilizer recommendation
- Irrigation scheduling

## Features
- AI disease detection from leaf images
- Crop prediction based on soil nutrients
- Fertilizer suggestions
- Irrigation schedule
- Multilingual support

## Technologies Used
- React.js
- Flask
- Python
- TensorFlow
- MongoDB

## Installation
1. Clone the repository.
2. Install dependencies.
3. Start the frontend and backend.

## Usage
Open the application, enter the required details, and view AI recommendations.
